# tooltips-pro
Add class 'tooltips-pro' and attribute 'data-tooltips-title' to use it
Example: 
```
<a href="javascript:void(0)" data-tooltips-title="Add" target="_blank" class="tooltips-pro">
    <i class="fa fa-plus-circle" aria-hidden="true"></i> Add
</a>  
```
