# tooltips_pro
Add class 'tooltips_pro' and attribute 'data-tooltips-title' to use it
Example: 
```
<a href="javascript:void(0)" data-tooltips-title="Add" target="_blank" class="tooltips_pro">
    <i class="fa fa-plus-circle" aria-hidden="true"></i> Add
</a>  
```
